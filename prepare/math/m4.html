    • 1.
    • 2.
    • 3.
    • 4.
    Hint

    Let the five consecutive even numbers be

    x, x + 2, x + 4, x + 6 and x + 8 respectively.

    According to the question,

    x + x + 2 + x + 4 + x + 6 + x + 8

    = 5 × 52

    ⇒ 5x + 20 = 260

    ⇒ 5x = 260 – 20

    ⇒ x = = 48

    E = x + 8 = 48 + 8 = 56

    ∴ B × E = 50 × 56 = 2800
